# SteamAPI_C

##### Prerequisites (Linux):

* GCC (8.3+)
* CMake (3.13.4+)
* OpenSSL (1.1.1b+)
* CURL (7.64.0+)
* Json-C (0.12.1+)

##### Linux:
    cd build
    cmake ..
    make
    cd ../bin
    ./steam_api

##### Example installation packages (Ubuntu 18.04):
    sudo apt-get install libssl-dev
    sudo apt-get install libcurl4-openssl-dev
    sudo apt-get install libjson-c-dev
